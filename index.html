自動區分地震波形中的 P 波與 S 波¶

林芳伃 汪其璇 黎昕雯

摘要¶

本次專題，我們希望利用STA/LTA 方法，挑選可能的地震事件(P、S波的到時)。 訓練MLP (Multi-Layer Perceptron)，用於自動區分地震波形中的 P 波與 S 波，來實現自動挑波。

正文¶

原始目標¶

最初我們原本想透過使用現有的深度學習模型（如 PhaseNet-DAS 等），自動偵測並標記 DAS 數據中的 P 波和 S 波到達時間，以提升地震監測的效率，以利於地震定位，來了解臺北盆地西側的地質。期望可與中央氣象局的地震資料（2019.10 以後資料缺少規模2以下的地震事件）進行比對，期望能從DAS資料中找出中央氣象局有列出來的大地震事件以及規模較小的區域性地震。最後，可以利用cluster 或 K-mean 等工具來了解地震分群與該區域地質之間的關聯(如 Mildon et al.(2022) 和 Sun et al.(2024))。

修改目標¶

但為更好的實現與環境資料處理課堂中的連結，我們自己去訓練模型，來實現自動挑波。接著受限於DAS 數據缺少已挑選的P、S波到時資料，我們改地震儀而非DAS做為資料屬裡的數據。有由於原先設定的目標北台灣西側(位於臺北盆地與林口台地間)，缺少區域性地震事件，我們決定使用20180206位於台灣東側花蓮規模6.26的地震事件及其前震與餘震事件，對模型進行訓練。

圖 1: 20180206花蓮地震事件地震報告。(交通部中央氣象局地震測報中心提供)

討論¶

根據結果，可以獲得本模型在自動區分地震波形中的 P 波與 S 波方面表現尚可，整體分類準確率約為 65% 至 71%。測試準確率（64.69%）與驗證準確率接近，表明模型在不同數據集上的泛化能力尚可，但仍有改進空間。特徵分析顯示，Envelope 等關鍵特徵對分類決策具有重要影響，模型成功捕捉了背景波形與 P/S 波之間的特徵差異。然而，模型對背景波形的分類較穩定，但對 P 波與 S 波的判別自信度偏低，可能受限於數據分布或特徵提取不足。此外，驗證損失略高於訓練損失，且訓練準確率高於測試準確率，顯示模型對訓練數據的依賴程度較高，存在一定的過擬合現象。以下為細項的可改善部分：

數據¶

a.擴充訓練與驗證數據集，增加 P 波與 S 波的標籤數量，並確保數據分布均衡，特別是在稀有波形事件的標注上。

b.新增非地震事件之波形資料與標籤，訓練模型挑波能力。

模型¶

a.引入更具表現力的模型架構（如 CNN、LSTM），進一步捕捉波形的時序特徵與非線性關係。

b.使用正則化（如 Dropout 或 L2 正則化）來降低過擬合現象，提升泛化能力。

其他Features¶

探索新的時域與頻域Features（如小波變換、頻譜特徵）來進一步增強模型表現。

結論¶

模型在地震波形分類中的初步表現具備一定實用價值，但在準確性與穩定性方面仍需進一步優化。未來可考慮改進特徵提取方法、平衡數據分布或引入更為複雜的模型架構（如深度學習），以提升對 P 波與 S 波的判別能力。同時，需加強正則化技術以緩解過擬合現象，進一步提升模型的泛化能力，從而實現更精確的地震監測效能。

# 時窗處理：根據波相選擇不同的時間窗
def preprocess_waveform(waveform,PZ):
    waveform.detrend(type="demean")
    waveform.detrend(type="linear")
    waveform.taper(max_percentage=0.05, type="hann")

    nqf=waveform.stats.sampling_rate/2 ###nyquist frequency
    pre_filt = [0.005, 0.008 , nqf-2, nqf] ### zero phase filter band
    ### 把PZ的資料餵給waveform
    attach_paz(waveform,PZ )
    paz = dict(waveform.stats.paz)
    waveform.simulate(paz_remove=paz,pre_filt=pre_filt)
    # waveform.plot()
    waveform.taper(0.05, type='hann')
    waveform.filter("bandpass", freqmin=0.1, freqmax=20.0)
    return waveform

# 特徵提取函數
def extract_features(waveform):#, phase):
    data = waveform.data
    env = envelope(data)
    sta_lta = classic_sta_lta(data, int(0.1 * waveform.stats.sampling_rate), int(1 * waveform.stats.sampling_rate))

    features = [
        np.max(env), np.mean(env), np.std(env),
        np.max(sta_lta), np.mean(sta_lta),
    ]
    return features

def extract_data_with_station_time(files, selected_phases, event_info, tt_ps_df):
    X, y = [], []
    for file in files:
        st = read(file)

        for tr in st:
            station = tr.stats.station
            comp = tr.stats.channel
            starttime = tr.stats.starttime
            endtime = tr.stats.endtime

            PZ_pattern = f"SAC_PZs_TW_{station}_{comp}_*"
            PZ = glob.glob(PZ_pattern)[0]

            tr_processed = preprocess_waveform(tr,PZ)

            for event_id in event_info["event_id"]:
                event_date = event_info[event_info["event_id"] == event_id]["event_date"].values[0]
                event_time = event_info[event_info["event_id"] == event_id]["event_time"].values[0]
                event_mag = event_info[event_info["event_id"] == event_id]["mag"].values[0]
                # event_lon = event_info[event_info["event_id"] == event_id]["lon"].values[0]
                # event_lat = event_info[event_info["event_id"] == event_id]["lat"].values[0]

                event_time_str = f"{event_date}{str(event_time).zfill(8)[:6]}"
                event_time_dt = datetime.strptime(event_time_str, "%Y%m%d%H%M%S")
                event_timestamp = event_time_dt.timestamp()

                if not (starttime.timestamp <= event_timestamp <= endtime.timestamp):
                    continue
                # 不是每個地震事件都能被測站偵測到!
                matching_rows = tt_ps_df[
                    (tt_ps_df["station"] == station) &
                    (tt_ps_df["phase"].isin(selected_phases)) &
                    (tt_ps_df["event_id"] == event_id)
                ]

                # sta_lon = station_info[station_info["station"] == station]["longitude"].values[0]
                # sta_lat = station_info[station_info["station"] == station]["latitude"].values[0]

                if matching_rows.empty:
                    continue
                else: # 如果事件被測站偵測到再進行處理
                  start_time = event_time_dt - timedelta(seconds=10)
                  end_time = event_time_dt + timedelta(seconds=120)

                  P_arrival = matching_rows[matching_rows["phase"] == "P"].iloc[0]["travel_time"]
                  S_arrival = matching_rows[matching_rows["phase"] == "S"].iloc[0]["travel_time"]

                  try:
                      # 根據波相進行時間窗截取，再進行前處理
                      tr_processed_evt = tr_processed.copy()
                      tr_processed_evt.trim(starttime=UTCDateTime(start_time), endtime=UTCDateTime(end_time))
                      # print(f"event")
                      # tr_processed_evt.plot()

                      for phase in selected_phases:
                          if phase == "P":
                            # trim tr_processed from P_arrival to P_arrival+5
                            tr_trimmed = tr_processed_evt.trim( 10+P_arrival, 10+P_arrival+5)
                            # print(f"P")
                            # tr_trimmed.plot()
                            features = extract_features(tr_trimmed) # , phase)
                            phase_label = 0

                          else:
                            # trim tr_processed from S_arrival to S_arrival+15
                            tr_trimmed = tr_processed_evt.trim(10+S_arrival, 10+S_arrival+15)
                            # tr_trimmed.plot()

                            features = extract_features(tr_trimmed) # , phase)
                            phase_label = 1

                          X.append(features,event_mag)
                          y.append(phase_label)
                  except Exception as e:
                      print(f"波形處理失敗: {e}")
                      continue
    return np.array(X), np.array(y)

損失值：¶

驗證損失值（約0.8）比訓練損失值（約0.5）高，進一步佐證了輕微過擬合現象。

測試準確率：¶

測試準確率（64.69%）與驗證準確率幾乎一致，顯示模型在測試資料上的表現穩定，未有顯著的過擬合或數據洩漏問題。

總結：¶

模型能夠區分 P 波與 S 波，但分類準確率約為 65%-71%，表現尚可但仍有改進空間。 可能的改進方法：

  1. 增加數據量或調整特徵提取方式。
  2. 使用更複雜的模型結構（如 LSTM 或 CNN）捕捉時序特徵。
  3. 探索更適合的正則化或超參數調整來減少過擬合現象。

def extract_features(steam): #, phase):
    features = []
    for trace in steam:
      channel = trace.stats.channel
      data = trace.data
      env = envelope(data)
      sta_lta = classic_sta_lta(data, int(0.1 * trace.stats.sampling_rate), int(1 * trace.stats.sampling_rate))

      if "HLR" in channel: # R comp -> 1
          features.extend([np.max(env), np.mean(env), np.max(sta_lta), np.mean(sta_lta),1])
      elif "HLT" in channel or "HLZ" in channel: # TZ comp -> 2
          features.extend([np.max(env), np.mean(env), np.max(sta_lta), np.mean(sta_lta),2])

      features = [
          np.max(env), np.mean(env),
          np.max(sta_lta), np.mean(sta_lta),
      ]
    return features

def extract_data_with_station_time( stations, dates, selected_phases, event_info, tt_ps_df):

        sta_lon = station_info[station_info["station"] == station]["longitude"].values[0]
        sta_lat = station_info[station_info["station"] == station]["latitude"].values[0]
        for date in dates:
            st_files = glob.glob(f"TW.{station}.*{date}*")
            if not st_files:
              print(f"找不到檔案: {station} - {date}")
              continue
            st = Stream()
            for tr_file in st_files:
              tr = read(tr_file)[0]
              st.append(tr)

              PZ_pattern = f"SAC_PZs_TW_{station}_{tr.stats.channel}_*"
              PZ = glob.glob(PZ_pattern)[0]
              attach_paz(tr,PZ)
            print(st)

            stf = st.copy()
            starttime = stf[0].stats.starttime # UTCDateTime
            endtime = stf[0].stats.endtime # UTCDateTime

            for event_id in event_info["event_id"]:
                  event_date = event_info[event_info["event_id"] == event_id]["event_date"].values[0]
                  event_time = event_info[event_info["event_id"] == event_id]["event_time"].values[0]
                  event_mag = event_info[event_info["event_id"] == event_id]["mag"].values[0]
                  event_lon = event_info[event_info["event_id"] == event_id]["lon"].values[0]
                  event_lat = event_info[event_info["event_id"] == event_id]["lat"].values[0]

                  event_time_str = f"{event_date}{str(event_time).zfill(8)[:6]}"
                  event_time_utc = UTCDateTime(event_time_str)

                  if not (starttime <= event_time_utc <= endtime):
                      continue

                  # 不是每個地震事件都能被測站偵測到!
                  matching_rows = tt_ps_df[
                      (tt_ps_df["station"] == station) &
                      (tt_ps_df["phase"].isin(selected_phases)) &
                      (tt_ps_df["event_id"] == event_id)
                  ]

                  dist, az, baz = gps2dist_azimuth(event_lat, event_lon, sta_lat, sta_lon)
                  stf.rotate(method="NE->RT",back_azimuth=baz)

                  stf.detrend(type="demean")
                  stf.detrend(type="linear")
                  stf.taper(max_percentage=0.05, type="hann")

                  nqf=(stf[0].stats.sampling_rate)/2 ###nyquist frequency
                  pre_filt = [0.005, 0.008 , nqf-2, nqf] ### zero phase filter band
                  paz = dict((stf[0].stats.paz))
                  stf.simulate(paz_remove=paz,pre_filt=pre_filt)
                  stf.taper(0.05, type='hann')
                  stf.filter("bandpass", freqmin=0.1, freqmax=20.0)
                  stf_ori = stf.copy()
                  if matching_rows.empty:
                      continue
                  else: # 如果事件被測站偵測到再進行處理
                    start_time = event_time_utc - 10

                            y.append(phase_label)
                    except Exception as e:

    return np.array(X), np.array(y)
